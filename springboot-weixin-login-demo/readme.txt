
第三方微信登录小案例
##########################################
1. nginx中进行配置(我使用的windows版本的)
	server {
	        listen       80;
	        server_name  localhost;
	
	        location /{
	        	# 注意端口是本地程序跑起来之后的端口
	            proxy_pass http://127.0.0.1:8080;
	        }
	}
	
##########################################	
2. 本地host文件中配置:

127.0.0.1 www.txjava.cn

##########################################
3. 启动程序就可以访问了,访问地址为:

www.txjava.cn/login.html
